## Setup environment
#Install Dependencies
```bash
npm install
```
#Install playwright browser
```bash
npx playwright install chromium
```

## Running Tests
#Run all tests
```bash
npm test
```

#Run Tests in Headed Mode (visible browser)
```bash
npm run test:headed
```

#Run Tests with UI Mode (interactive)
```bash
npm run test:ui
```

## View Test Report
```bash
npm run report
```
